# <旅スポ>

## サイト概要
### サイトテーマ
ユーザー同士が持っている観光スポット情報を共有するサイト


### テーマを選んだ理由
自分自身がバイクで旅行をする上で、事前に旅行先の情報を集めてルート決めをしているが、
様々なサイト・雑誌・SNSから情報集めをする必要がある。

情報を一括でまとめられれば情報集めの手間が省けると思った為。
また、再度同じ場所を訪れる際に過去の情報を再確認したい場面がある為。


### ターゲットユーザ
・旅行を計画しているユーザ
・既に旅行先にいるユーザ
・車、バイク、徒歩などの様々な移動手段のユーザ


### 主な利用シーン
・個人での旅行ルート決め
・グループ内での旅行ルート情報の共有
・旅行の備忘録
・旅行計画を立てる際の情報収集場所
